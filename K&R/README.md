# K&R Exercises

This directory contains my solutions to the exercises from *The C Programming Language* by Brian W. Kernighan and Dennis M. Ritchie. It does not contain all exercises.

The primary goal is learning and understanding the language through practice.

Most solutions are developed independently as part of the learning process. AI is used sparingly, mainly for clarification, discussion, or verification. For some particularly challenging exercises, a greater degree of AI assistance may be used.
